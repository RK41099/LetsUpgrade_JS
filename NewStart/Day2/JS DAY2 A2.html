Javascript Essentials : String and Arrays

String : There are 3 methods in JS to get Substring : 1.substring 2.substr 3.slice

1. str.slice(start, end) :
Returns the part of the string from start to (but not including) end
    let str = "stringify";
    console.log( str.slice(0, 5) ); : Output -> strin
console.log( str.slice(2) ); : Output -> ringify (everything from position 2)
console.log( str.slice(-4, -1) ); Output -> gif

_____________________________________________

2. str.substring(start, end) :
Returns the part of the string between start and end. This is almost the same as slice, but it allows start to be greater than end.
    let str = "stringify";
    console.log( str.substring(2, 6) ); : Output -> ring
console.log( str.substring(6, 2) ); : Output -> ring
console.log( str.substring(2) ); : Output -> ringify (everything from position 2)
Negative positions in substring method returns an empty string

_____________________________________________

3. str.substr(start, length) :
Returns the part of the string from start with the given length. In contrast with the previous methods, this one allows us to specify the length instead of the ending position:
console.log( str.substr(2, 4) ); Output -> ring
console.log( str.substr(-4, 2) ); Output -> gi

**Sort Method for Sorting of Elements**
    let arr = [ 1, 2, 15 ];
    // the method reorders the content of arr
    arr.sort();
    alert( arr ); // 1, 15, 2

The items are sorted as strings by default.
Literally, all elements are converted to strings for comparisons.
For strings, lexicographic ordering is applied and indeed "2" > "15". To use our own sorting order, we need to supply a function as the argument of arr.sort(). The function should compare two arbitrary values and return:

function compareNumeric(a, b) {
if (a > b) return 1;
if (a == b) return 0;
if (a < b) return -1;
}

let arr5 = [ 1, 15, 2 ];
console.log(arr5);
arr5.sort(compareNumeric);

console.log(arr5); // 1, 2, 15
